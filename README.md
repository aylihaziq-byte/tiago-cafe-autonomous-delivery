# TIAGo Café Autonomous Mobile Robot Delivery Stack

This repository contains a functional ROS package implementing multi-point autonomous navigation, a thread-safe custom graphical user interface, and an integrated computer vision verification system using a TIAGo robot within a simulated café environment.

Execute the master automation script to clear caches, launch the Gazebo simulation, synchronize laser frames, and open the UI dashboard:
